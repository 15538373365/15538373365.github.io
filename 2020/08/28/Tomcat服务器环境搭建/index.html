Tomcat服务器环境搭建

  • 2020-08-28

前置条件

配置了JAVA_HOME变量

环境搭建

  1. 官网下载tomcat
  2. 解压到任意英文目录
  3. 使用管理员身份 运行dos命令行 , 通过dos命令行进入解压后的tomcat\bin路径
  4. 在dos命令行中 输入安装指令:
    service install
  5. 在dos命令行中 观察安装成功的提示:
    The service ‘tomcatx’ has been installed

修改端口号

  1. 打开tomcat\conf\server.xml文件
  2. 寻找到大概在70行的一个 8080文字, 修改为 8090
  3. 打开tomcat\bin\tomcat8w.exe –> start

卸载tomcat

tomcat的卸载指令
1.    使用管理员身份 运行dos命令行 , 通过dos命令行进入解压后的tomcat\bin路径
2.    在dos命令行中 输入卸载指令:
    service remove
windows的删除服务命令
在任意路径下, 使用管理员身份打开dos命令行, 输入:
sc delete 服务名
例如:    sc delete tomcat8

开发环境搭建

1.打开eclipse
2.window
3.preferences
4.server
5.Runtime Environments
6.Add
7.选择Apache下对应版本,并勾选Create a new local server
8.选择安装的目录
9.选择jre
10.finish

http://ip地址:端口号/项目名/文件名.后缀
